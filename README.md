# Animal Lifespan Predictor
A machine learning-based web application that predicts the lifespan of animals using features like speed and weight. The project uses Linear and Polynomial Regression models and is deployed using Streamlit.

## Features
- Predict animal lifespan based on input features
- Supports:
  - Linear Regression
  - Polynomial Regression
- Interactive and user-friendly interface
- Real-time predictions

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Pickle
- 
## How It Works
1. User enters animal speed and weight  
2. Selects the prediction model  
3. Model processes the input  
4. Displays predicted lifespan
